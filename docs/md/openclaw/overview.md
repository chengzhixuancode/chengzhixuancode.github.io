# OpenClaw - 开源 Claude AI 客户端体验

> OpenClaw 是一个开源的 Claude AI 客户端，提供图形界面，适合那些不习惯 CLI 的开发者使用。本文记录我的体验感受。

---

## 什么是 OpenClaw

OpenClaw 是一个第三方开源的 Claude 桌面客户端，特点：

- 🖥️ **图形界面** - 相比 Claude Code 的 CLI 更直观
- 🔄 **多会话管理** - 方便管理不同项目的对话
- 💾 **本地存储** - 对话记录保存在本地
- 🔑 **自己的 API Key** - 使用你自己的 Anthropic API Key

## 为什么要用 OpenClaw

| 特性 | Claude Code (官方 CLI) | OpenClaw (开源 GUI) |
|------|------------------------|---------------------|
| 界面 | 终端命令行 | 图形界面 |
| 代码操作 | 直接读写文件 | 需要复制粘贴 |
| 开源 | 闭源 | ✅ 开源 |
| 价格 | API 收费 | API 收费 + 免费客户端 |

如果你更喜欢图形界面，或者只是想和 Claude 对话而不直接修改代码，OpenClaw 是不错的选择。
